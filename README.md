# KR_WaterDepthAnalysis
R을 활용한 대한민국 동해, 서해, 남해 수심 데이터 시각화

<h2>🌎︎ 출처 : GEBCO, JOISS(관할해역해양정보 공동활용시스템</h2><br><br>

<br>
<strong>👩🏻‍💻 팀구성</strong>
- 개인 프로젝트<br><br>

<strong>🔎 개요</strong>
- 대한민국 해안 수심 시각화
: 대한민국 각 해안의 수심을 비교해 볼 수 있는 서비스<br><br>


<strong>📌 목적</strong>
- 대한민국의 지리적 특성에 따른 해안 수심을 시각화된 자료로 비교할 수 있도록 하고자 함<br><br>


<strong>💻 개발환경</strong>
- 프로젝트 도구 : Rstudio
- 사용 언어 : R<br><br>

<strong>📝 학습목표</strong>
- R 기초 문법 숙지
- 외부 데이터 활용
- 데이터 시각화(plot)
